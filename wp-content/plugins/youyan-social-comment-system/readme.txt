= 什么是友言？ =
国内最大的分享工具JiaThis旗下的社会化评论工具，一键评论同步至社交网络，增加网站用户活跃度，调动好友参与评论，有效提升网站社会化流量！

= 为什么使用友言？ =
1. 精确定位用户：不再是冷冰冰的用户名与邮箱，用户授权后，即可看到丰富多彩的社会化网络评论。
2. 高质量评论：实名制的留言系统，大大提高评论的质量。
3. 智能统计：智能的数据统计功能，让您完全掌控网站的信息动向。
4. 更多流量与访客：通过社会化媒体优化(SMO)，吸引更多网站回访，提升网站PV流量，口碑营销更加高效。

= 功能特色？ =
1. 数据互通：数据双向同步，只要设置密钥和提交同步请求，数据就可以实时同步了。
2. SNS同步：博客文章自动同步至绑定微博，微博的评论自动同步至网站。
3. 强大的数据管理后台，轻松处理海量评论信息；严密的评论审核机制，将评论分为正常，待审核，垃圾评论，已删除。
4. 黑名单和敏感词过滤系统，有效拦截网络垃圾评论。
5. 实用热榜插件：全站最热、最新评论；全站最热、最新文章插件。
6. 丰富DIY自定义：不再受限于系统默认，facebook、disqus风格任你选择；智能转发图片；匿名登录还可以玩隐身哦！
7. 表情、个人信息、社区激发用户评论热情，调动网站积极性，人性化选择，真实性传播。
8. 新评论邮件提醒，随时倾听用户的声音！


友言的成长离不开广大用户的支持，友言团队与您共创互联网最优质的产品！

== Installation ==
= 在线安装： =
1. 在Wordpress安装插件库搜索“uyan”，下载并启用插件。
2. 第一次登陆需要注册账号。

= 下载安装：= 
1. 下载文件"youyan-social-comment-system"，解压缩到WordPress根目录wp-content/plugins 2. 进入wordpress管理后台，按照提示操作，第一次登陆需要注册账号。

== Frequently Asked Questions ==

= 1. “友言”的目标是什么？ =
一个简单而强大的社会化评论及聚合平台，为广大网站提供最优质的服务，实现社会化网络优化(SMO)，提升网站社会化流量。

= 2. 友言提供给什么样的用户使用？ =
网站主通过社会化评论框，增加用户粘度，同时也让网站链接分享到互联网各个角落。个人用户可以查看自己在互联网上的所有评论、喜欢自己的评论数、社区评论情况。

= 3. 如果长期不登陆友言，评论数据会不会被删除？
所有注册友言并已产生社会化评论的网站，其评论数据会永久保存。

= 4. 友言社会化评论代码，安全吗？ =
友言是通过OAuth 2.0系统来实现，服务器多层数据加密权限验证机制，以及多层硬件和软件的多重审核防护，不会泄露任何用户隐私，防护墙保护性很高，可以防止任何恶意攻击，全方位保证用户的使用安全。

= 5. ”友言“参照的服务是什么？ =
来自 "Facebook comment box" 和 "Disqus Comment System"

= 6. 为什么站长需要在网站根目录下放置代理文件(proxy.html) =
“友言”通过html的iframe标签嵌入网页中，为了能自动调整框架高度（当用户添加和删除评论时），我们需要站长在网站的根目录下放置这样一个代理文件。（下载链接在友言主页 uyan.cc，或是wordpress的管理界面中）

= 7. 能否提供技术帮助？ =
您可以咨询：友言客服邮箱 help@uyan.cc   客服QQ ：1735067958

= 1.8 =
1）我们修复了所有已知的兼容性问题，现在在所有的浏览器，IE6到9，搜狗，360，Chrome, Firefox, Safari，都能够完全正确的运行。
2）站长配置登陆选项和顺序。
3）邮件提醒
4）全面更新的统计系统。
我们对用户评论和回访做了全面的分析和呈现。现在，您能够从“用户”，“评论”，“社交网络”三个维度检测到自己网站的评论和回访趋势。在您的网站上，某个用户有多少条评论，带来多少回访。每种社交网络有多少条评论，带来多少回访。乃至于具体到每一条评论带来多少回访，都能够精确的呈现。您可以很容易的发现本网站的核心活跃用户。
5）更好的用户评论管理系统。
我们现在支持以页面为单位进行评论与管理。同时，有的网站有多名编辑，但只有一名站长。站长则可发放某一页面的管理权限，生成一个只有当前页面权限的链接给编辑。
6）更好的“顶”和“踩”。
支持对本篇文章的“顶”和“踩”，“顶”之后用户可选直接分享文章。对文章和评论的“顶”和“踩”再不需要用户登陆。可以登陆，也可以在未登录时进行（用cookie判断最近进行）
7）评论社区。，能看每一个用户的最新评论以及活跃社区。帮助用户建立评论记录，帮助站长建立评论社区。
8）自定义违章标题，图片，等
我们默认的同步信息样式为：
我才在“  <网站名>（<网站域名>）”的页面《<页面标题>》发表了评论: “<用户评论>” http://t.cn/ar2GGH <图片>
其中网站名在站长的管理界面里设置。网站域名自动提取。网站标题为为网页的 “title” 标签，图片默认为页面中第一个<img>标签。
站长可以自己配置想要的标题和图片。只需在网页里加入如下内容：
<meta name=  “UYPageTitle” content=  “ 您设置的微博转发文章标题” />
<meta name=  “UYImg” content=  “ 您设置的微博转发图片” />
9）多页面共享评论框
友言默认取页面URL作为一组评论的标识。如果站长想要多个页面（有不同的URL）共享一个评论框，可以在这些页面里指定：
<meta name=  “UYId” content=  “ 您设置的页面ID” />
设为相同ID，则可共用一个评论框。
10）(wordpress插件专有)，导入导出数据到友言中。站长可以点击多次导入数据，每次只有新评论（如果有的话）才会被导入（同理，导出也是）
强烈建议站长选择导入现有wordpress数据，然后禁用原有wordpress评论系统，以使界面美观。
11）(wordpress插件专有)，文章发微博。
当站长绑定自己的新浪或者腾讯的微博账号后，在文章发布和更新的界面会分别出现下面的选择框。默认同步，若站长不勾选，则自动用站长账号为文章发微博。
12）(wordpress插件专有)，已有“文章微博”的文章下评论处理。
用户在页面内的评论，将以自动转发并评论站长的文章微博的形式进行。
13）(wordpress插件专有)，配置站长自己的APPKEY
您不光可以用友言的APPKEY，也可以用自己的APPKEY
14）其他
我们还有十多处各种小的调整或升级，如站长可自定义匿名评论用户的默认URL，更快的加载速度，非常稳定的社交网络分享（迄今位置我们测试数百起，没有失败过一起）。我们的保留特色：十几项站长的自定义属性，和自定义CSS等。
